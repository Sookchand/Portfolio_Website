Project Overview

I developed an interactive AI-driven system to provide supermarkets with actionable insights and optimize key operations, including sales forecasting, customer segmentation, and personalized recommendations.

The system utilizes a modular architecture with specialized AI agents and a Streamlit interface for user interaction and dynamic report generation. Users can ask questions in natural language and receive detailed analyses, visualizations, and actionable recommendations.

The main dashboard of the Supermarket Optimization System

Key Features

AI Agent Architecture

The system incorporates four AI agents to provide a comprehensive suite of analytical and decision-support capabilities:

Explanation Agent

Provides natural language explanations of model predictions and insights, enhancing user understanding. This leverages my expertise in Natural Language Processing (NLP).

Recommendation Agent

Generates actionable business recommendations based on data analysis, such as optimizing inventory or tailoring marketing campaigns. This showcases my ability to translate data into strategic guidance.

Simulation Agent

Simulates different business scenarios (e.g., price changes, marketing campaign impacts) and predicts outcomes, enabling proactive decision-making.

Orchestration Agent

Coordinates the specialized agents and provides a unified interface, demonstrating my skills in system architecture and modular design.

Visualization Capabilities

The system generates a range of visualizations to support data-driven decision-making:

Customer Segment Comparison

Visualizes customer segments based on key metrics (rating, sales, etc.).

Sales Analysis

Shows the impact of different factors on sales and revenue.

Time-Based Analysis

Visualizes patterns in customer behavior by time of day.

Rating Factors Analysis

Identifies factors affecting customer ratings.

Streamlit Interface

A user-friendly Streamlit interface allows users to interact with the AI agents using natural language queries, making the system accessible to both technical and non-technical users.

The natural language query interface of the application
